## Deskripsi Pelatihan

Pelatihan automated testing untuk aplikasi web menggunakan Playwright, mencakup functional testing, page object pattern, dan CI/CD integration.

## Tujuan Pelatihan

Setelah mengikuti pelatihan ini, peserta akan mampu menguasai konsep dan praktik automated testing dengan playwright.

## Target Peserta

**Cocok untuk:**
- QA Engineer
- Test Automation Engineer
- Developer
- DevOps Engineer

**Skill prerequisite:**
Basic programming (JavaScript/TypeScript), basic web development, HTML/CSS understanding

## Durasi Pelatihan

**2 hari (16 jam)**

## Topik Pelatihan

1. Playwright fundamentals dan architecture
2. Browser automation (Chromium, Firefox, WebKit)
3. Test framework dengan Playwright Test
4. Database testing dan setup
5. Parameterized test data
6. Locator strategies dan best practices
7. Test scenarios: normal dan error cases
8. Generate test data dengan Faker
9. Form validation testing
10. Page Object Pattern implementation
11. Parallel test execution
12. CI/CD integration
13. Visual regression testing

## Technology Stack

Playwright, TypeScript/JavaScript, Faker.js, CI/CD tools
