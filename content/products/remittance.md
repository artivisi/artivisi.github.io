## Deskripsi Produk

Aplikasi yang memudahkan pengiriman uang secara cepat dan aman, baik domestik maupun internasional. Solusi ini menyederhanakan transfer keuangan dengan biaya minimal dan proses yang efisien, memungkinkan pengguna mengirim uang kepada keluarga dan teman dengan mudah dan terjangkau.

## Fitur Utama

- Transfer remittance domestik dan internasional
- Integrasi payment gateway
- Tracking real-time
- Biaya kompetitif
- Proses yang efisien dan aman

## Spesifikasi Produk

- **Ukuran Kode:** 1024 baris
- **Ukuran JAR:** 175MB

## Teknologi yang Digunakan

**Framework & Bahasa:**
- Spring Boot 3.2.5
- Java 21

**Database:**
- PostgreSQL
- FlywayDB untuk migrasi

**Frontend & Templates:**
- Thymeleaf dengan Layout Dialect, Extras, Spring Data, dan Security extensions

**Security:**
- Spring Security framework
- JWT authentication via JJWT libraries

**Libraries Tambahan:**
- Gson untuk data parsing
- UA-Parse untuk user-agent detection
- Spring Boot Starter Validation

## Development Stack

- Maven build tool dengan Spring Boot plugin
- Lombok untuk code generation
- Spring Boot Starter Test framework
- Git version control
