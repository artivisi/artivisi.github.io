## Deskripsi Pelatihan

Pelatihan Git dari fundamental hingga advanced workflows, branching strategies, collaboration patterns, dan CI/CD implementation dengan GitLab.

## Tujuan Pelatihan

Setelah mengikuti pelatihan ini, peserta akan mampu menguasai konsep dan praktik git & ci/cd: version control & collaboration.

## Target Peserta

**Cocok untuk:**
- Developer (all roles)
- DevOps Engineer
- Technical Lead
- Project Manager

**Skill prerequisite:**
Basic command line usage, basic Git knowledge (add, commit, push)

## Durasi Pelatihan

**2 hari (16 jam)**

## Topik Pelatihan

1. Git fundamentals dan internal concepts
2. Git reset, revert, dan cherry-pick
3. Branching strategies (feature, topic, release)
4. Merge vs Rebase
5. Release management workflows
6. Collaboration workflows (Centralized, Feature Branch, Gitflow, Forking)
7. CI/CD concepts dan implementation
8. Automated integration testing
9. GitLab CI/CD pipeline
10. Git patch management (apply, am)

## Technology Stack

Git, GitLab, GitLab CI, Docker (for CI/CD)
