## Naming Convention

- Use lowercase letters
- Use hyphens (-) to separate words
- Use .png extension
- Keep names concise and descriptive

Examples:
- ✓ `bca.png`
- ✓ `bank-mandiri.png`
- ✓ `bea-cukai.png`
- ✗ `BCA_Logo.png`
- ✗ `Bank Mandiri Logo.PNG`

## Image Requirements

- **Format:** PNG with transparent background
- **Max height:** 200px (for optimal display in carousel)
- **Quality:** High resolution preferred, will be scaled down for web
- **Color mode:** RGB

## Usage

Client logos are displayed in:
1. Homepage client section (`layouts/home.html`)
2. Consulting page carousel (`layouts/consulting/consulting.html`)

Path format: `/img/clients/filename.png`
